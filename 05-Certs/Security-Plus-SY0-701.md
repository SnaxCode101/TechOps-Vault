# 🎓 CompTIA Security+ (SY0-701) — Study Index

## Exam Overview
| Property | Detail |
|---|---|
| Exam Code | SY0-701 |
| Questions | Maximum 90 questions |
| Duration | 90 minutes |
| Passing Score | 750 / 900 |
| Format | Multiple choice + Performance-Based Questions (PBQs) |
| Validity | 3 years |
| Prerequisites | Recommended: Network+ and 2 years IT experience |

## Domain Breakdown (SY0-701)

| Domain | Weight | Topic |
|---|---|---|
| 1.0 | 12% | General Security Concepts |
| 2.0 | 22% | Threats, Vulnerabilities, and Mitigations |
| 3.0 | 18% | Security Architecture |
| 4.0 | 28% | Security Operations |
| 5.0 | 20% | Security Program Management and Oversight |

---

## Domain 1 — General Security Concepts (12%)

**Key Topics:**
- Security controls (technical, managerial, operational, physical)
- Control categories (preventive, detective, corrective, deterrent, compensating)
- CIA Triad → CIA-Triad
- AAA framework (Authentication, Authorization, Accounting)
- Cryptography basics → Cryptography-Basics
- PKI, certificates, certificate authorities
- Password concepts: salting, hashing, MFA
- Zero Trust architecture

---

## Domain 2 — Threats, Vulnerabilities, and Mitigations (22%)

**Key Topics:**
- Threat actors (nation-state, APT, insider threat, hacktivist, script kiddie)
- Attack types:
  - Social engineering (phishing, vishing, smishing, pretexting, baiting)
  - Malware types (ransomware, RAT, worm, trojan, rootkit, keylogger, spyware, adware)
  - Application attacks (SQLi, XSS, CSRF, SSRF, buffer overflow, race condition)
  - Network attacks (DoS/DDoS, MITM, replay, session hijacking, ARP spoofing, DNS poisoning)
  - Password attacks (brute force, dictionary, credential stuffing, rainbow table)
- OWASP Top 10 → OWASP-Top-10
- Vulnerability scanning vs penetration testing
- CVE, CVSS scoring
- Threat intelligence (STIX, TAXII, IOC)

---

## Domain 3 — Security Architecture (18%)

**Key Topics:**
- Cloud security models (IaaS, PaaS, SaaS)
- Shared responsibility model
- Network segmentation (VLAN, DMZ, air gap, jump server)
- Firewall types (packet filter, stateful, NGFW, WAF)
- IDS vs IPS (signature-based, anomaly-based, heuristic)
- VPN (IPSec, SSL/TLS VPN, split tunnel)
- Proxy servers, load balancers
- Data loss prevention (DLP)
- Endpoint security (EDR, XDR, antivirus, HIDS)
- Identity and Access Management (IAM)

---

## Domain 4 — Security Operations (28%)

**Key Topics:**
- Incident response lifecycle (Preparation, Identification, Containment, Eradication, Recovery, Lessons Learned)
- Digital forensics (chain of custody, order of volatility, disk imaging)
- Log analysis and SIEM
- Network traffic analysis → Wireshark
- Vulnerability management lifecycle
- Penetration testing phases (Reconnaissance, Scanning, Exploitation, Post-exploitation, Reporting)
- Identity management (SSO, LDAP, SAML, OAuth, OIDC)
- Access control models (DAC, MAC, RBAC, ABAC)
- Hardening techniques (patching, disabling services, changing defaults)
- Backup and recovery (3-2-1 rule)

---

## Domain 5 — Security Program Management and Oversight (20%)

**Key Topics:**
- Risk management (risk identification, assessment, treatment: accept/avoid/transfer/mitigate)
- Risk types (inherent, residual, control)
- Frameworks: NIST CSF, ISO 27001, SOC 2, PCI-DSS, HIPAA
- Business continuity (BCP) vs disaster recovery (DRP)
- RTO (Recovery Time Objective) vs RPO (Recovery Point Objective)
- Data classification (public, internal, confidential, restricted/top secret)
- Privacy regulations (GDPR, CCPA)
- Third-party risk management
- Security awareness training
- Policy types (AUP, password policy, BYOD, change management)
