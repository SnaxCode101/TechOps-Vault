## What This Covers

This micro-certification validates competency in ServiceNow Flow Designer — the platform's primary no-code/low-code automation tool.

**Core competencies demonstrated:**

- Building and configuring Flows in Flow Designer
- Working with Triggers (record-based, scheduled, REST API, inbound email)
- Using Flow Logic — If/Else, For Each, Wait for Condition, Try/Catch
- Configuring Actions — Create Record, Update Record, Send Email, Run Script, Call Subflow
- Understanding Data Pills and variable passing between steps
- Building and invoking Subflows
- Testing and debugging via Flow Execution History
