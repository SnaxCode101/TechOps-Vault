## What This Covers

This micro-certification validates foundational knowledge of the ServiceNow platform — its architecture, navigation, core applications, and purpose as an enterprise service management platform.

**Core competencies demonstrated:**

- ServiceNow platform overview — cloud-based, single-data-model architecture
- Core applications: ITSM, ITOM, HR, CSM, SecOps, AppEngine
- Navigation: Application Navigator, Favorites, History, global search
- Forms, lists, and record management basics
- ServiceNow instances: development, test, production (instance lifecycle)
- Users, groups, and roles — foundational access control model
- Now Platform UI (Next Experience / Polaris UI)
- ServiceNow's place in enterprise IT: ITSM, workflow, automation

## Platform Fundamentals Quick Reference

| Concept | Detail |
|---|---|
| Instance URL | `https://[instance].service-now.com` |
| Application Navigator | Left sidebar — filter box to find any module |
| Tables | All data in ServiceNow stored in tables (e.g., `incident`, `change_request`) |
| Records | Each row in a table is a record |
| Forms | Single-record view with fields |
| Lists | Multi-record view; sortable, filterable |
| Roles | Grant access to modules and data (`itil`, `admin`, `sn_incident_write`) |
| Groups | Collections of users; used for assignments and approvals |
| Update Sets | Package of customizations for promotion between instances |
