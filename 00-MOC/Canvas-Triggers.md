# Canvas Triggers

> Master list of entry-point trigger questions for the SecOps-Vault Canvas.
> Every trigger maps to a parent flow → subflow → output on the canvas.
> Organized from mundane (daily) to sophisticated (advanced ops).

---

- "I forgot what this port number is"
- "I need to remember command syntax"
- "I need to look up an acronym or definition"
- "I'm taking notes during class right now"
- "I need a quick cheat sheet"
- "I need a command reference right now"
